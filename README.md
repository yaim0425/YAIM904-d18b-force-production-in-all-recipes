## üì¶ Description

**YAIM904 d18b ‚Äì Force Production in All Recipes** allows **productivity modules to affect all recipes**, including those that are normally excluded in vanilla Factorio.

This MOD removes standard restrictions and forces productivity bonuses to apply universally, enabling higher output from every production chain.

It is especially useful for megabases, experimental factories and players who want maximum production efficiency.

## ‚öôÔ∏è How it works (Player view)

* Productivity bonuses apply regardless of vanilla restrictions
* No configuration or manual changes are required
* Works automatically once the MOD is enabled

Production mechanics remain the same, except that productivity is always allowed.
